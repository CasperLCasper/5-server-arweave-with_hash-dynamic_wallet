About Platform

Wallet Visualizer transforms your wallet into unique visual art and lets you save it as an NFT with verifiable authenticity — permanently stored on Arweave.

Connect your wallet, pick a theme, and the platform generates your personalized wallet visualization — an image and a 15-second video.

What data generates the image?

The visualization is created using your real blockchain wallet data:

  • 💰 Balance — determines particle size and movement intensity
  • 🪙 Token count — influences particle quantity and connection density
  • 🔄 Transaction count — sets visualization complexity and speed
  • 🔑 Wallet address — serves as a unique seed for unrepeatable patterns
  • 🎨 Chosen theme — determines color palette and visual style

How it works?

  1. Connect wallet — use any Web3 wallet (MetaMask, Enkrypt, etc.)
  2. Choose blockchain — multiple chains supported for data reading
  3. Click "Generate NFT" — platform creates image, video, and metadata
  4. Receive files — all files saved to your computer as a ZIP archive
  5. Files stored on Arweave — permanent, decentralized, pay-once-store-forever
  6. Minting on Base — artwork is officially minted as NFT with SHA256 hash recorded on-chain

What do you get?

  • 🖼️ Image — wallet visualization in PNG format
  • 🎬 Video — 15-second animation
  • 📄 Metadata — JSON file with NFT description and attributes
  • 📦 ZIP archive — all three files at once on your computer
  • 🔵 NFT on Base — digital artwork recorded directly on Base network
  • ⛓️ SHA256 hash on-chain — mathematically verify file authenticity anytime
  • 🐘 Arweave Permanent Storage — data stored forever on the permaweb

Why Base network and hash matter?

Each NFT gets a SHA256 hash — a unique digital fingerprint, all on Base:

  • Low cost & speed: Thanks to Base (Ethereum L2), NFT minting is lightning-fast with minimal gas fees
  • Hash is permanently recorded in blockchain smart contract
  • Anyone can verify if a file exactly matches the original
  • Even if storage services change, the hash remains as immutable proof of ownership

Are files secure?

  • 💻 Saved on your computer — immediately after creation
  • 🐘 Uploaded to Arweave — permanent, decentralized storage on the permaweb
  • 🛡️ Protected by hash on Base — any file is mathematically verifiable on the secure Base blockchain
